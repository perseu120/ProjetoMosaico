## 1. Fluxo de Cadastro do Participante

Este fluxo descreve o processo pelo qual um participante se cadastra na plataforma, desde a inserção dos dados até a confirmação do cadastro, incluindo a validação dos dados e o registro de auditoria.

## 2. Fluxo de Realização de Check-in

Este fluxo detalha como um organizador gerencia os participantes inscritos e realiza o check-in, com as respectivas checagens de permissão e condições.

## 3. Fluxo de Emissão de Certificado

Este fluxo detalha como o participante acessa e baixa seu certificado após o evento, condicionado à confirmação de presença.
